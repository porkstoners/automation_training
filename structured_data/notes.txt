  - YAML

  - NETCONF
    - Uses XML

  - JSON

- Why is structured data useful 
 - Allows us to easily to transform them into python Dictionaries
 - This is important because it means we dont have to scan large blobs of text and scan for specific content. 
 - It allows us to programtically access the data and return data accurately
